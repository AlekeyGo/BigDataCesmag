# BigDataCesmag

Repositorio para el material de clases de la especialización en Big Data de la Universidad CESMAG.

Este repositorio centraliza apuntes, presentaciones, ejercicios, datasets y prácticas usadas en las asignaturas de la especialización. Está organizado por módulos o materias para facilitar el acceso y la colaboración entre estudiantes y docentes.

- **Objetivo:** Guardar y compartir el material docente de la especialización en CESMAG.
- **Contenido:** Apuntes, diapositivas, soluciones, notebooks, datasets y recursos adicionales.
- **Estructura:** Cada asignatura o módulo tiene su propia carpeta con el material correspondiente.
- **Uso:** Recurso interno para estudiantes y profesores; respetar derechos de autor y las políticas de la universidad antes de redistribuir.
